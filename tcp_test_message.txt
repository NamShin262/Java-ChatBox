TCP TEST FILE
================

Mục đích:
- Dùng để test chức năng gửi file văn bản qua TCP.
- Kiểm tra bên nhận có hiển thị đúng nội dung hay không.

Nội dung mẫu:
1. Xin chào từ file test TCP.
2. Đây là dòng văn bản thứ hai.
3. TCP phù hợp với truyền dữ liệu tin cậy và đúng thứ tự.

Ký tự đặc biệt:
- Tiếng Việt: ă â ê ô ơ ư đ
- Số: 1 2 3 4 5
- Ký hiệu: ! @ # $ % ^ & *

Kết thúc file test TCP.
